# @pathscale/fonts-sansation

## Installation

    npm i @pathscale/fonts-sansation

## Usage

import from js

```JS
import '@pathscale/fonts-sansation'
```

declare what font to use in css

```CSS
body {
    font-family: 'sansation';
    font-weight: bold;
}
```

## Variants supported:

* regular
* regular italic
* lighter
* lighter italic
* bold
* bold italic

# @pathscale/fonts-studio-revenge

## Installation

    npm i @pathscale/fonts-studio-revenge

## Usage

import from js

```JS
import '@pathscale/fonts-studio-revenge'
```

declare what font to use in css

```CSS
body {
    font-family: 'studio-revenge';
    font-weight: bold;
}
```

## Variants supported:

* regular
* regular italic
* lighter
* bold

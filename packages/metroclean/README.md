# @pathscale/fonts-metroclean

## Installation

    npm i @pathscale/fonts-metroclean

## Usage

import from js

```JS
import '@pathscale/fonts-metroclean'
```

declare what font to use in css

```CSS
body {
    font-family: metroclean;
    font-weight: regular;
}
```

## Variants supported:

* regular
* medium
* extra-bold
